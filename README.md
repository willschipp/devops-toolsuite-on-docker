# toolsuite on docker

- collection of agile project support tools deployed into docker
- considerations include
  - SCM
  - Project Management
  - CI
  - Chat
  - Deliverables/Binaries


## Tools


Tool Name | Role
----------|-----
gogs | SCM
taiga | project management
jenkins | CI
rocket.chat | chat
sonatype nexus | CD
minio | Binaries
